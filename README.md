# D&D Майстерня — Створення персонажів

Локальний застосунок для створення та ведення персонажів D&D 5e. Раніше був побудований на Base44
(хмарна автентифікація + база даних), тепер повністю автономний:

- **Вхід не потрібен** — застосунок відкривається одразу, для одного користувача.
- **Персонажі зберігаються локально**: у зібраному `.exe` — у JSON-файлі (через Electron), у режимі
  `npm run dev` в браузері — у localStorage.
- Дані про класи та раси D&D 5e підвантажуються з відкритого [Open5e API](https://api.open5e.com) —
  докладні описи там англійською мовою (AI-переклад і генерація зображень, які раніше залежали від
  Base44, прибрані).

## Запуск у режимі розробки (у браузері)

```bash
npm install
npm run dev
```
Відкрийте адресу, яку виведе Vite (зазвичай http://localhost:5173).

## Збірка десктопного застосунку (.exe / .dmg / .AppImage)

```bash
npm install
npm run dist:win     # Windows: dist/D&D Майстерня Setup *.exe
npm run dist:mac      # macOS: dist/*.dmg
npm run dist:linux    # Linux: dist/*.AppImage
```

Готовий інсталятор з'явиться в папці `dist`.

### Де зберігаються дані персонажів

У встановленому `.exe`-застосунку персонажі зберігаються у звичайному JSON-файлі за шляхом, який
Electron надає через `app.getPath('userData')` — зазвичай:
```
C:\Users\<Користувач>\AppData\Roaming\D&D Майстерня\characters.json
```
Резервне копіювання — просто скопіюйте цей файл. Його можна навіть відкрити текстовим редактором.
